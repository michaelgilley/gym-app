GYM PROGRAM

4-Week Progression Plan

Wk 1:

Learn movements. Pick weights where last 2 reps are hard but form stays clean.

Wk 2:

Add 5 lbs to barbell lifts. Go up one dumbbell size where possible.

Wk 3:

Same as week 2. If you hit top of rep range easily, add weight.

Wk 4:

Add a 4th set to the main compound each day. Push intensity.

Day A - Upper Push (Monday, ~65 min)
Warmup
5 min cardio Light intensity
Arm circles / Shoulder rotations 2 min
Warm-up sets on bench 2 x 10
Compound
Barbell bench press 3 x 8-10 2-3 min rest
Accessories
Incline dumbbell press 3 x 10-12 90s
Overhead dumbbell press 3 x 10-12 90s
Cable lateral raise 3 x 12-15 60s
Tricep cable pushdown 3 x 12-15 60s
Cooldown
Chest doorway stretch 45s
Cross-body shoulder stretch 45s each
Overhead tricep stretch 30s each
Bicep wall stretch 30s each
Child's pose 60s
Ab Finisher
Goblet overhead leg raise 3 x 10-12 60s
Plank 3 x 30s
Bicycle crunches 3 x 15 each

Active Rest Days

  • Walks: 30-45 min, 3x/week, brisk but conversational
  • E-bike: 30-60 min, 1-2x/week, moderate effort
  • Avoid: Long rides before lower body days

Nutrition Guidelines

  • Protein: 0.8-1g per pound bodyweight daily
  • Calories: 200-300 below maintenance (recomp)
  • Timing: Don't skip meals on gym days
